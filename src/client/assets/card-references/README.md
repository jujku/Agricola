# Card Reference Sheets

This directory stores the supplied minor improvement and occupation card reference sheets.

Runtime UI imports cropped art from `src/client/assets/minor-improvements/` and `src/client/assets/occupations/`; these reference sheet images remain source/reference material and should not be imported directly by runtime code.

Expected files:

- `minor-improvements-sheet-01.png`: first supplied sheet, minor improvement cards.
- `occupations-sheet-01.png`: second supplied sheet, occupation cards.
- `occupations-sheet-02.png`: third supplied sheet, occupation cards.

When the original image files are available locally, place them here using the exact filenames above.
